一、React 优化

绑定事件时使用 evt.target.dataset 代替工厂函数

尽量避免例子一，改用例子二的写法

例子一：
handleClick = (index) => {
  return () => {
    console.log(index)
  }
}

{
  list.map((obj, index) => {
    return <div key={index} onClick={this.handleClick(index)}>index</div>
  })
}
例子二：
handleClick = evt => {
  console.log(evt.target.dataset.index)
}

{
  list.map((obj, index) => {
    return <div data-index={index} key={index} onClick={this.handleClick}>index</div>
  })
}

二、Props

PropTypes类型检查:

import PropTypes from 'prop-types';

class testComponent extends Component {
  render () {
    return (
      <p>{this.props.text}</p>
    )
  }
}

testComponent.propTypes = {
  text: PropTypes.string.isRequired, //text类型为字符串，且必传
  children: PropTypes.element.isRequired, //表明testComponent使用时内部必须嵌套react元素
}

PropTypes和TS的type check（下面简称TC）区别

  • TC是编译时检查，而PropTypes是运行时检查
  • TC更注重传参和返回值检测，而PropTypes则是对prop的限制
  • PropTypes缺少对null和undenfined的检查

PropTypes中可以使用的类型

defaultProps定义props的默认值:

testComponent.defaultProps = {
  text: 'hello world',
}
