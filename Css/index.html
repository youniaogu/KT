二、sticky（粘性）定位

sticky 在与 flex 一起使用时会失效(原因是 flex 在副轴上默认是 stretch，将副轴长度拉满，所以 sticky 会无效)，需要添加 align-items:flex-start。(来源：stackoverflow)

三、flex布局

flex属性由flex-grow(默认值为0)、flex-shrink(默认值为1)、flex-basic(默认值为auto)组成

flex属性可以指定一、二、三个值：

  1. 单个值，有三种情况：

    • 无单位的数字会被当作flex-grow处理

    • 有效的宽度值会被当作flex-basic处理

    • 关键字none，auto或initial.

  2. 两个值，第一个值必须为满足flex-grow的值（无单位数字），第二个值有两种情况：

    • 无单位的数字会被当作flex-shrink处理

    • 有效的宽度值会被当作flex-basic处理

  3. 三个值：

    • 即遵循flex-grow, flex-shrink, flex-basic的原则

flex-flow属性由flex-firection || flex-wrap组成
